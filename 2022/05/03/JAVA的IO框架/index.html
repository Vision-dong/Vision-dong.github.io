流的概念

概念：内存与存储设备之间传输数据的通道。

流的分类

按方向

输入流：将<存储设备>中的内容读入到<内存>中。

输出流：将<内存>中的内容写入到<存储设备>中。

按单位

字节流：以字节为单位，可以读写所有数据。

字符流：以字符为单位，只能读写文本数据。

按功能

字节流：具有实际传输数据的读写功能。

过滤流：在节点流的基础之上增强功能。
